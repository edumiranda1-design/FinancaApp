# Meu Dinheiro 💰

App simples de finanças pessoais, feito para qualquer pessoa usar sem dificuldade.

---

## O que é este app?

O **Meu Dinheiro** é um painel (dashboard) para acompanhar o saldo das suas contas mês a mês. Você digita quanto tem em cada conta, e o app mostra automaticamente como seu patrimônio está evoluindo ao longo do ano.

---

## Como usar

### 1. Lançar os saldos
- Use as setas **‹ ›** no topo para navegar entre os meses
- Digite o saldo atual de cada conta nos campos correspondentes
- Clique em **"Salvar este mês ✓"** para guardar os dados

### 2. Ver o resumo
- Toque na aba **Resumo** para ver o gráfico de evolução do patrimônio
- O app mostra automaticamente se você cresceu ou caiu em relação ao mês anterior

### 3. Definir metas
- Toque na aba **Metas**
- Digite sua **Meta financeira** e sua **Super Meta**
- Acompanhe a barra de progresso e receba feedbacks de incentivo

## Funcionalidades

- Navegação mês a mês pelo ano
- Cálculo automático do patrimônio total
- Comparação com o mês anterior (crescimento ou queda)
- Feedbacks simpáticos com emojis conforme seu progresso
- Gráfico de evolução anual do patrimônio
- Barra de progresso de metas com mensagens de encorajamento
- Dados salvos automaticamente no navegador (sem necessidade de cadastro)

---

## Informações técnicas

- **Tecnologia:** HTML, CSS e JavaScript puro
- **Armazenamento:** localStorage do navegador (os dados ficam salvos no seu dispositivo)
- **Dependência externa:** Chart.js 4.4.1 (gráficos)
- **Compatível com:** Chrome, Firefox, Safari, Edge — desktop e celular
- **Sem internet necessária** após o primeiro carregamento

---

## Observações importantes

- Os dados são salvos **somente no navegador** em que você está usando. Se trocar de navegador ou limpar os dados do navegador, os lançamentos serão perdidos.
- Recomenda-se usar sempre o mesmo navegador para manter o histórico.
- O app não envia nenhum dado para a internet.
